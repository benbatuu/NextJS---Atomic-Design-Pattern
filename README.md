## Getting Started

Clone the repo and run the following command to install dependencies:

```bash
npm install
# or
yarn install
```


Run the development server 🔥:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
